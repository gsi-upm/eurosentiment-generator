How to use the Eurosentiment Generator's API

To access the API just send a POST request to http://demos.gsi.dit.upm.es with two parameters:

  • template: The ID of the desired template.
  • document: Document to translate.

On GNU/Linux, you can test the the API using curl. A request would look like this:

curl -F"template=PT_HA_to_Marl" -F"document=@PT_HA_example.csv" http://demos.gsi.dit.upm.es/eurosentiment/generator/process {% endblock %}
